# neusfeet

A pet client for https://newsapi.org

### Building

1. Add `api_key` to your `local.properties` file like
    ```     
    api_key="your api key"
    ```
    You can get it [here](https://newsapi.org)
2. Build it!

### Based on

1. Kotlin
2. Koin DI
3. Retrofit
4. RxJava
5. Android ViewModel
6. Room
7. Paging
8. Navigation
